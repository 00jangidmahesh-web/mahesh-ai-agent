# Mahesh AI Assistant

AI Voice Assistant built using:
- OpenAI
- LangChain
- Streamlit
- RAG
- FAISS
- Whisper
- OpenAI TTS

## Features
- Voice Input
- AI Voice Response
- Conversational Memory
- RAG Retrieval
- Streamlit UI

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
